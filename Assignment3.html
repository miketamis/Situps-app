Assignment 3.

Emulator

For the emulator I went really simple and didn't even use any 3rd party libraries. The emulator is 13 lines of Javascript and some CSS and HTML. The emulator simply creates an IFrame and loads the app of choice into the iframe. Their are 4 Buttons to interface with the app. The events for these buttons are sent through to the iframe using the PostMessage function of web browsers.

Prototype

The prototype is a very simple Situp Trainer App. I completed most the task I set out the complete, and didn't end up making any change to the original design. I didn't get time to add in using the Gyroscope and Accelerometer to count the situps for you, but completed everything else I set out to complete.
